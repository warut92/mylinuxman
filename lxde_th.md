# LXDE  ภาษาไทย

[Github lxde](https://github.com/lxde)

## แปลแล้ว

lxappearance
LXDE GTK+ theme switcher

lxsession
The classic LXDE session manager

lxterminal
VTE terminal emulator written in GTK

lxtask
Mirror of LXTask: LXDE task manager.

lxmusic

lxlauncher
LXDE launcher for netbooks (recovered copy)

lxinput
LXDE keyboard and mouse configuration (recovered copy)

lxhotkey

lxde-common
LXDE common configuration files (recovered copy)

gpicview

lxrandr

pcmanfm

lxpanel

lxde-icon-theme
LXDE standard icon theme (recovered copy)

menu-cache

lx-control-center
