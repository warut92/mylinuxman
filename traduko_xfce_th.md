# สถาพแวดล้อมเดสก์ท็อป Xfce
เป็นเดสก์ท็อปลำดับที่สอง ที่ผมได้เริ่มในการใช้ระบบปฏิบัติการลินุกซ์ ซึ่งอันแรกคือ Unity ที่มีอยู่ใน Ubuntu ณ ขณะนั้น จากนั้นผมก็มาลอง Xfce และ Lxde แต่ว่า Xfce จะทำงานได้ลื่นไหลมากที่สุดกับโน้ตบุ๊คเครื่องเก่าของผมในขณะนั้น

## Xfce ในภาษาไทย

เป็นการต่อยอดจากคุณ theppitak โดยใช้ข้อความแปลเดิมของท่านในการแปลต่อ ซึ่งมาจาก suggestion หรือจาก deleted resource บ้าง จากการเปรียบเทียบการแปลของโปรแกรมอื่นบ้าง ซึ่งท่านแปลไว้ได้อย่างดีแล้ว 

## แนวการแปล

เนื่องจากผมยึดแนวภาษาจากการแปลเดิม ภาษาที่ใช้อาจจะดูเก่า โบราณไปหน่อย แต่คิดว่าเป็นการแสดงตัวตน เอกลักษณ์ มีความเป็นมา ความเป็นอิสระจากระบบปฏิบัติค่ายใหญ่ อาจจะไม่เป็นมิตรกับผู้ใช้มากนั้น เป็นคำที่ไม่คุ้นเคย แต่คำเหล่านั้นก็ไม่ได้มากนัก (ผมไม่รู้ว่าผู้ใช้คนไทย ใช้ภาษาไทยในระบบปฏิบัติการลินุกซ์มากน้อยเพียงใด จากประสบการณ์อันน้อยนิด มักเห็นใช้ในภาษาอังกฤษ) และอีกอย่าง ถ้าจะอัปเดตคำเก่าเหล่านั้น ก็สามารถทำได้ในอนาคต หากใครมีเครื่องไม้เครื่องมือครบ

## รูปแบบการแปล

Failed to ... >> ...ไม่สำเร็จ

Unsupported ... >> ...ไม่รองรับ

## คำที่น่าสนใจ 
parse > แจง

file > แฟ้ม

paste > แปะ

reset > คืนกลับ, ล้างค่า 

notifications > การแจ้งเหตุ

minimized > ย่อเก็บ

theme > ชุดตกแต่ง

appearance > รูปลักษณ์

thumbnail > ภาพย่อ

instance > อินสแตนซ์

link > จุดเชื่อม

location > ตำแหน่ง

launcher > ตัวเรียกโปรแกรม, ตัวเรียกแอปพลิเคชัน, ปุ่มเรียก

Empty Trash > เทถังขยะ

pane > ช่อง

emblems > ตรา

intermediate file > ไฟล์ชั่วคราว

Last Modified > เวลาแก้ไข

## ไม่แน่ใจ?

Disc images > อิมเมจแผ่น เทียบจาก ISO images > อิมเมจ ISO (xfburn)

Pulse Audio Volume Control > ช่องควบคุมเสียง Pulse Audio (xfce4-mixer)

## ข้อสังเกต

บางคำแปลไม่สอดคล้องกัน เช่น

desktop บางทีใช้ พื้นโต๊ะ บางที่ใช้การทับศัพท์ เดสก์ท็อป
[คิดว่าใช่!: พื้นโต๊ะ หมายถึง บริเวรหน้าจอที่สามารถสร้างทางลัดต่าง ๆ ได้ แต่ เดสก์ท็อป สิ่งแวดล้อมพื้นโต๊ะ (Desktop Environment) ซึ่งจะประกอบไปด้วยชุดซอฟต์แวร์ต่าง ๆ]

การแปลส่วนใหญ่เป็นการแปลโดยไม่เน้นการทับศัพท์

Synchronize > การปรับให้ตรงกัน

type > บางที่ ชนิด บางที่ รูปแบบ
